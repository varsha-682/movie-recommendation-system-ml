

# 🎬 Movie Recommendation System (Machine Learning)

A simple Movie Recommendation System built using Python and Machine Learning techniques like **CountVectorizer** and **Cosine Similarity**.

---

## 📌 Objective

To recommend movies similar to a given movie using **content-based filtering**.

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

---

## ⚙ Algorithm Used

* CountVectorizer
* Cosine Similarity

---

## 🚀 How It Works

1. Load movie dataset
2. Preprocess the data
3. Extract features (genres, keywords, cast, crew etc.)
4. Convert text data into vectors using **CountVectorizer**
5. Calculate similarity using **Cosine Similarity**
6. Recommend top **5 similar movies**

---

## 🎥 Example

**Input:** Avengers

**Output:** Similar movies like

* Iron Man
* Captain America
* Thor
* Guardians of the Galaxy
* Doctor Strange

---

## 📊 Applications

* OTT Platforms (Netflix, Amazon Prime etc.)
* E-commerce product recommendation
* Music streaming apps
